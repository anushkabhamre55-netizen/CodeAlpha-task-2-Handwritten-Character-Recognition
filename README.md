# Handwritten Character Recognition Using Machine Learning

## Project Overview

This project recognizes handwritten digits (0–9) using Machine Learning techniques. A Random Forest Classifier is trained on the Digits Dataset from Scikit-Learn and deployed using Streamlit for real-time digit prediction.

## Objective

The objective of this project is to develop a machine learning model capable of identifying handwritten numerical digits accurately.

## Technologies Used

* Python
* Scikit-Learn
* NumPy
* Pandas
* Streamlit
* Google Colab
* VS Code

## Dataset

The project uses the Digits Dataset available in Scikit-Learn.

### Dataset Information

* Total Samples: 1797
* Number of Features: 64
* Classes: 10 (Digits 0–9)
* Image Size: 8 × 8 Pixels

## Machine Learning Algorithm

### Random Forest Classifier

Random Forest is an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

## Project Workflow

1. Load Dataset
2. Preprocess Data
3. Train-Test Split
4. Train Machine Learning Model
5. Evaluate Accuracy
6. Save Trained Model
7. Create Streamlit Application
8. Predict Handwritten Digits

## Model Performance

* Accuracy: Approximately 97%–99%
